# task-contract

This packaged Skill must mirror `skills/task-contract/SKILL.md`.

Use this skill to clarify, bound, and contract a user request before execution.

Required behavior:

- Auto-Skeleton first.
- Optimized Task.
- Output Contract.
- Decision Points when choices matter.
- Approval Gate for high-impact work.
- Loop Contract Mode for bounded iterative work.
- Loop Log when iteration is used.
- One clear next step.
