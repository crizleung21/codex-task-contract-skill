# Decision Point Patterns

Use Decision Points when multiple valid interpretations exist and choosing one would change the work.

## Pattern

```md
## Decision Point A — [Decision Name]

| Option | Choice | Trade-off |
|---|---|---|
| 1 | ... | ... |
| 2 | ... | ... |
| 3 | ... | ... |

Recommended: Option 2, because ...

Reply template:

```text
A = 2
```
```

## Rules

- Prefer numbered options.
- Keep options mutually distinct.
- Explain trade-offs briefly.
- Recommend a default when there is a clear best option.
- Avoid open-ended questions unless necessary.
