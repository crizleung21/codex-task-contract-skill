# Loop Contract Template

Use this template when a task needs bounded iteration.

## Loop Contract Fields

| Field | Content |
|---|---|
| Loop Objective | Measurable goal. |
| Loop Type | Simple / Coding-debug / Research / Documentation / Repo maintenance / High-impact. |
| Iteration Unit | Smallest useful action per cycle. |
| Observation Method | Evidence checked after each cycle. |
| Adjustment Strategy | How the next step changes. |
| Validation Method | How completion is verified. |
| Stop Condition | What ends the loop. |
| Max Iterations | Maximum cycles. |
| Escalation Trigger | When to ask the user. |
| Approval Gate | Whether execution may proceed. |
