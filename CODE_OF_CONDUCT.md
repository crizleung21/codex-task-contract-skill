# Code of Conduct

This project expects respectful, constructive, and professional collaboration.

## Expected Behavior

- Be respectful.
- Keep feedback specific and actionable.
- Discuss ideas and implementation details clearly.
- Help keep the project safe and useful.

## Unacceptable Behavior

- Harassment.
- Personal attacks.
- Discriminatory language.
- Spam.
- Deliberately unsafe contributions.

## Enforcement

Maintainers may moderate discussions and contributions to protect the project and its community.
