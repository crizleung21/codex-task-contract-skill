# Usage

## Explicit Invocation

```text
Use $task-contract to clarify this task first.
```

## Compact Contract

Use for small low-risk work.

Sections:

1. Auto-Skeleton
2. Optimized Task
3. Output Contract
4. Next Step

## Full Contract

Use for complex, ambiguous, or high-impact work.

Sections:

1. Auto-Skeleton
2. BLUF
3. Optimized Task
4. Assumptions
5. Constraints
6. Decision Points
7. Output Contract
8. Execution Plan
9. Acceptance Criteria
10. Risk Gate
11. Next Step

## Loop Contract

Use for iterative work that needs observation, adjustment, validation, and stop conditions.
